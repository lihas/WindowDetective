Defining Custom Window Data

Various parts of Window Detective can be customised by editing files in the user appdata directory. The following sections list each asset under that directory, what they are used for, and how to modify or add to them.

In Windows XP, the user data directory is: C:\Documents and Settings\<username>\Application Data\Window Detective\
In Vista and Windows 7, it is: C:\Users\<username>\AppData\Roaming\Window Detective\
