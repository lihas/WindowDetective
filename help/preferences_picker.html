Picker Preferences

Picker

Hide main window while picking

The main Window Detective pane is temporarily hidden so that you can easily see and pick any windows beneath it. If this is unticked, the main pane will stay visible at all times.

Window Info

You can choose what will be displayed in the info pane when using the picker. The following options are available:

  • Window Class
  • Text/Title
  • Position
  • Size
  • Dimensions
  • Handle
  • Parent Handle

Window Highlight

These options control the highlight that is drawn over a selected window.

Style The highligh can either be a border around the window or completely filling the window. If it is a border, the thickness can be set.
Colour Specifies the colour and transparency of the highlight. When using the filled style, the transparency is automatically set to a sensible value so that the highlighted window can still be seen under the highlight. However, you can set it to whatever you like.
