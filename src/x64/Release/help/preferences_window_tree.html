Window Tree Preferences

Hidden Windows

Grey hidden windows

Any windows that are not visible on screen (that is, they do not have a WS_VISIBLE style), will be shown as grey in the tree. Note: The tree must be refreshed for this to take effect.

Change Highlighting

When an item in the tree is changed, it will be highlighted.

Duration The time that the item will stay highlighted for
Colour Choose colours for each highlight event. The immediate colour is the colour of the item that is updated. The unexpanded colour is the colour of the closest visible (expanded) ancestor of an item which is not expanded.

Note: When an item is changed, not all of it's properties (in the tree and any property windows) will not get automatically updated.
