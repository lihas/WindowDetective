Window Class Icons

Directory: <Window Detective App Data>\data\window_class_icons\

Image files under this directory are used by Window Detective as icons for window classes. These icons are shown in the window tree.

You can add images for custom window classes, or to override Window Detective's image for system classes. The name of the file must be the same as the name of the window class you wish to use it for (case-sensitive), and the format must be either PNG or ICO.
