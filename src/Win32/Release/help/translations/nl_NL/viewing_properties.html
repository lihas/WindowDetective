Venster Eigenschappen Bekijken

Je kunt de eigenschappen van een venster bekijken door rechts te klikken op dat venster in de boom en "Bekijk Eigenschappen" te selecteren. Er zal een venster getoont worden met diverse eigenschappen voor het geselecteerde venster.
Deze eigenschappen houden in:
Tekst/Titel Toont de tekst voor de control/venster. Dit is de titel voor vensters en dialoog boxen.
Handle De handle (HWND) van het venster.
Ouder Handle Handle van het bovenliggend venster.
Afmetingen De x,y positie, breedte en hoogte van het venster.
Positie De x,y positie van het venster op het scherm. Gemeten in pixels waar 0,0 de linkerbovenhoek van het scherm is.
Grootte Breedte en Hoogte in pixels
Relatieve Afmetingen Als het venster een kind is van een ander venster, dan is dit de x,y positie en grootte ten opzichte van zijn ouder.
Client Afmetingen De afmetingen van het 'client' gebied van het venster. Dit is het gebied binnen de titelbalk en de rand.
Alleen toepasbaar op vensters en dialoog box controls.
Stijl bits De gecombineerde stijlbits van het venster als een integer, in hexadecimaal.
Stijlen Een lijst van stijlen toegepast op het venster.
Uitgebreide Stijl bits De gecombineerde uitgebreide stijlbits van het venster als een integer, in hexadecimaal.
Uitgebriede Stijlen Een lijst van uitgebriede stijlen toegepast op het venster.
Proces Eigenaar De bestandsnaam van het proces dat het venster heeft aangemaakt.
Proces Eigenaar ID Het huidige PID van de proces eigenaar.
Eigenaar Thread ID Het ID van de thread binnen het eignaar proces die het venster heeft aangemaakt.
Venster Eigenschappen Een lijst van applicatie-gedefinieerde venster eigenschappen. Elke eigenschap bestaat uit een string naam en een handle naar enige data. Kijk op MSDN voor meer informatie over venster eigenschappen.

In toevoeging op de venster eigenschappen, worden de eigenschappen voor de klasse van het venster ook getoont.
Deze eigenschappen houden in:
Klasse Naam Een string naam die de klasse identificeert.
Klasse Extra Bytes Het aantal bytes voor extra data van de klasse. Kijk op MSDN voor meer informatie over extra klassen en venster geheugen.
Venster Extra Bytes Het aantal extra venster bytes
Achtergrond Brush Handle voor de brush (HBRUSH) die gebruikt wordt om de achtergrond van het venster te tekenen. Dit kan een systeemgedefinieerde constante zijn die een systeemkleur representeert.
